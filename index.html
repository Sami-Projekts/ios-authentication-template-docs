üì± What you'll do:

üîß Detailed Instructions:

Common Issues & Solutions

üö´ Google Sign-In Not Working

Symptoms: Google Sign-In button doesn't respond or shows error

Solutions:

  • ‚úÖ Verify REVERSED_CLIENT_ID in URL schemes matches GoogleService-Info.plist
  • ‚úÖ Ensure GoogleService-Info.plist is added to project target
  • ‚úÖ Check Google Sign-In is enabled in Firebase Console
  • ‚úÖ Test on physical device (not simulator)
  • ‚úÖ Verify Bundle ID matches exactly between Xcode and Firebase

üçé Apple Sign-In Failing

Symptoms: Apple Sign-In shows error or doesn't appear

Solutions:

  • ‚úÖ Check "Sign in with Apple" capability is added in Xcode
  • ‚úÖ Verify App ID is configured in Apple Developer Portal
  • ‚úÖ Test on physical device with valid Apple ID
  • ‚úÖ Ensure proper provisioning profile is selected
  • ‚úÖ Check Apple Sign-In is enabled in Firebase Console

üìß Email Verification Issues

Symptoms: Verification emails not received or links don't work

Solutions:

  • ‚úÖ Check spam/junk folder in email client
  • ‚úÖ Verify Firebase email templates are properly configured
  • ‚úÖ Check authorized domains in Firebase Console
  • ‚úÖ Ensure Firebase project is properly set up
  • ‚úÖ Test with different email providers (Gmail, iCloud, etc.)

üî• Firebase Connection Issues

Symptoms: App crashes on startup or Firebase functions don't work

Solutions:

  • ‚úÖ Verify GoogleService-Info.plist is properly added to project
  • ‚úÖ Check Bundle ID matches exactly in all configurations
  • ‚úÖ Ensure Firebase.configure() is called in AppDelegate
  • ‚úÖ Verify Firebase products are properly imported
  • ‚úÖ Check Xcode console for specific Firebase error messages
