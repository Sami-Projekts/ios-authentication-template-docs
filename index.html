🎯 Getting Started

🎉 Two Ways to Get Started

Choose the approach that works best for you:

📱 Use Complete Template
🔧 Integrate into Existing Project

Option 1: Complete Template (Recommended)

Perfect for new projects or when you want a complete authentication system.

  1. Download the template and open in Xcode 15+
  2. Configure Firebase (see detailed setup below)
  3. Customize branding in AppConfig.swift and AppTheme.swift
  4. Run and test all authentication flows
  5. Build your app features on top of the authentication system

Option 2: Integrate into Existing Project

Add professional authentication to your existing iOS app.

  1. Copy core files (ViewModels, Models, Services, Utilities)
  2. Install dependencies (Firebase, GoogleSignIn)
  3. Integrate views into your app's navigation flow
  4. Configure Firebase and authentication providers
  5. Test integration and customize as needed

🛠️ Complete Setup Guide

Prerequisites

  • Xcode 15.0 or later
  • iOS 15.0+ deployment target
  • Swift 5.9 or later
  • Active Apple Developer Account
  • Google Developer Account (for Google Sign-In)

Step 1: Download and Open Project ⏳ Image loading...

Download the iOS Authentication Template and open it in Xcode. You'll see the complete project structure with all necessary files organized.

Xcode Project Overview - File structure and organization

Step 2: Add Swift Package Dependencies ⏳ Image loading...

First, we need to add the Firebase iOS SDK and Google Sign-In packages to your project.

Adding Package Dependencies in Xcode Project Settings

Step 3: Configure Firebase URL ⏳ Image loading...

Add the Firebase iOS SDK URL: https://github.com/firebase/firebase-ios-sdk

Entering Firebase iOS SDK repository URL

Step 4: Select Firebase Products ⏳ Image loading...

Choose the required Firebase products: FirebaseAuth, FirebaseFirestore, and FirebaseStorage.

Selecting Firebase products for authentication and data storage

Step 5: Add Google Sign-In SDK ⏳ Image loading...

Add the Google Sign-In iOS SDK: https://github.com/google/GoogleSignIn-iOS

Adding Google Sign-In iOS SDK to the project

Step 6: Create Firebase Project ⏳ Image loading...

Go to the Firebase Console and create a new project for your iOS app.

Firebase Console - Creating a new project

Step 7: Add iOS App to Firebase ⏳ Image loading...

Click "Add app" and select iOS. Enter your Bundle Identifier from Xcode.

Adding iOS app to Firebase project

Step 8: Configure Bundle Identifier ⏳ Image loading...

Enter your app's Bundle ID (e.g., com.yourcompany.authtemplate) exactly as it appears in Xcode.

Configuring Bundle Identifier in Firebase

Step 9: Download GoogleService-Info.plist ⏳ Image loading...

Download the GoogleService-Info.plist file from Firebase. This contains your app's configuration.

Downloading GoogleService-Info.plist configuration file

Step 10: Add plist to Xcode Project ⏳ Image loading...

Drag the GoogleService-Info.plist file into your Xcode project root, ensuring "Add to target" is checked.

Adding GoogleService-Info.plist to Xcode project

🎨 Easy Customization

App Configuration

Customize your app settings in AppConfig.swift:

struct AppConfig { // App Information static let appName = "Your App Name" static let appVersion = "1.0.0" static let bundleId = "com.yourcompany.yourapp" // Authentication Settings static let enableGoogleSignIn = true static let enableAppleSignIn = true static let enableEmailSignIn = true static let requireEmailVerification = true // UI Settings static let cornerRadius: CGFloat = 16 static let animationDuration: Double = 0.8 }
